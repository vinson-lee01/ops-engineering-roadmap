## 📌 使用建议

1. **Linux 基础** → 先刷 [Linux Journey](https://linuxjourney.com/) + [OverTheWire Bandit](https://overthewire.org/wargames/bandit/)
2. **Docker** → [Play with Docker](https://labs.play-with-docker.com/) 直接玩，不用本地装
3. **K8s** → [Play with K8s](https://labs.play-with-k8s.com/) 玩 5 节点集群，再跟 [Kubernetes The Hard Way](https://github.com/kelseyhightower/kubernetes-the-hard-way)
4. **Prometheus** → [PromLabs Playground](https://promlabs.com/prometheus-playground/) 练 PromQL
5. **综合提升** → [Killer Coda](https://killercoda.com/) 按场景练习

> 💡 **提示**：这些平台大多免费，但有些需要注册。建议用 GitHub 账号登录，比重新注册方便。
